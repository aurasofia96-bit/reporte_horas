# Problema 5 - Fundamentos de Programación

Solución en Python del problema 5 correspondiente a la Fase 5 del curso Fundamentos de Programación.

## Descripción
Este programa calcula el total de horas trabajadas semanalmente por cada recurso y clasifica su jornada laboral como:

- Sobretiempo
- Horario Estándar
- Inferior

## Lenguaje utilizado
- Python
